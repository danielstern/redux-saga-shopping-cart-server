About

A demo REST API to integrate with a Redux Saga shopping cart app.

Routes

route url about
User /user/:id Returns data on the given user ID
Cart /user/:owner Returns all items in user's cart, or an error if no cart exists.
Cart Add Item /cart/add/:owner/:itemID Adds an item to the specified owner's cart with the specified ID.
Cart Remove Item /cart/remove/:owner/:itemID Removes one unit of the specified item from the specified owner's cart.
Items /items/:ids Returns metadata for a collection of comma-separated item IDS.
Prices /prices/:symbol/:ids Returns the prices of the specified items in the specified symbol
Shipping /shipping/:ids Calculates the shipping costs for a block of item IDS.
Tax /tax/:symbol Returns the tax rate for the given currency or an error if one can't be found.
Checkout (TBC) /checkout/:owner Purchases all the items in the cart, or purchases none of the items in the cart if any errors occur.
Card (Validate) /card/validate/:owner Validates the owner's credit card - always returns {validated:true}
Cart (Validate) /cart/validate/:owner Validates the cart - returns {validated:true} if all cart items are in stock
Card (charge) /cart/charge/:owner Charges the user and purchases all items. Returns { success: true } if sufficient funds were available.
